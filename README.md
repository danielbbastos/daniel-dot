# daniel<span>.</span>

Personal portfolio of **Daniel Bastos** — frontend developer from Portugal, somewhere between code, design and product.

Built with [Next.js](https://nextjs.org) (App Router) and React, exported as a static site.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.

## Build

```bash
npm run build
```

Outputs a static export to `out/`, ready to host anywhere (e.g. GitHub Pages).

## Structure

- `app/` — routes, layout and global styles
- `components/` — shared React components
